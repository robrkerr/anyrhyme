GiftRAPPED API

GiftRAPPED is a rhyming dictionary that...

API Methods

Autocomplete

/autocomplete/string.json?limit=number

Example: /autocomplete/examp.json?limit=1

Match

/match/direction/with/exact/num_syllables/syllables/and/syllable1.json?limit=number

/match/direction/with/exact/num_syllables/syllables/and/syllable1/syllable2/syllable3.json?limit=number

Syllables have the format: onset,nucleus,coda

Onsets and codas have the format: phoneme1-phoneme2

Nuclei have the format: vowel-stress

Wildcards: *

Anti-match: ~

Example: /match/beginning/with/at-least/1/syllables/and/*,~aa1,*/s,ah0,m.json?limit=1
